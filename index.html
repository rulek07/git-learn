Hello Git

To Gallery
  1. Создаем папку с файлами
  2. Создаем "суппозиторий" в GitHub
  3. Можем внести необходимые изменения в файле
  4. Подлючаем файл к локальному реппозиторию git init (проверяем путь к папке в терминале, чтобы не получить потом 10000 файлов с изменениями в древе)
  5. Добавяем удаленный реппозиторий git remote add origin(путь реппозитория)
  6. Сохраняем изменения git add .
  7. Коммитим git commit -m "описание коммита"
  8. И теперь выгружаем на удаленный реппозиторий все изменения git push
  9. Создаем другую ветку под названием Second git checkout -b Second
  10. Git checkout (название ветки) для перехода на необходимую ветку
  11. Делаем дела на ветке и обводим мелом, сначала на второй, а потом на основной
  12. Теперь можно слить их воедино git merge second
  13. И запушить всё
  14. Использовал для тренировки сайт: https://learngitbranching.js.org/?locale=ru_RU
